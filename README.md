## 🚀 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/citizen-echo-ai.git
cd citizen-echo-ai
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Configure Environment Variables

Create a `.env.local` file:

```env
OPENAI_API_KEY=your_api_key_here
```

### 4. Run the App

```bash
npm run dev
```
